# VIBE CODING

A small static focus timer app built with HTML, CSS, and JavaScript.

## Structure

- `index.html`: app entrypoint
- `assets/css/app.css`: styles and skyline background
- `assets/js/app.js`: timer logic, color cycling, and bell audio
- `assets/js/helpers.js`: utility functions
- `config/site-config.json`: runtime site config
- `docs/notes.md`: project notes
- `archive/`: old and experimental files

## Run

Open `index.html` in a browser, or serve the folder with a local static server.
